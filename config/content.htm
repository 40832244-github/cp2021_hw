Set Up SSH

1.點選資料夾的Portablegit→bin→sh.exe

2.輸入ssh-keygen -t rsa -b 4096 -C "40832244"(需改成自己學號)

按enter→輸入/y/tmp/id_rsa→連續按按2次enter

3.下載putty_home、GitExtensions，將start_ipv4.bat移到白窗  並新增 

1
2
set GIT_HOME=%Disk%:\Portablegit\bin\
set GIT_SSH=%Disk%:\putty\plink.exe

4.打開.git資料夾，把config移到白窗，新增[remote "origin"]裡的url = git@github.com:40832244/cp2021.git

5.打開Putty Key Generator，將鼠標在紅框中搖晃會產生鑰匙，並先將它複製站存於筆記本

   完成後點選save public key(private 需多設一個密碼) 名為 id_rsa

6.打開Putty Configuration並照著圖片更改

7.點選SSH旁邊的加號點Auth，點Browse...點選檔案

選去剛剛儲存的id_rsa(記得選成所有檔案才看的到)

8.回到第6.部頁面點save，進到github網頁setting的SSH and GPG keys新增一把鑰匙，並到記事本內將剛剛在Putty Key Generator產生的鑰匙貼上，並檢查句尾是不是你的學號

點選Add SSH key

完成!!!
